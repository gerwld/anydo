ReactJS & Redux Anydo To-do list. Feel free to use it, change, or contribute.

### To run the project:
``` 
$ pnpm i
$ pnpm run dev
```

### Works currently:
1) CRUD oparations
2) localStorage cache (tasks, tags)
3) Axios + redux thunk logic for fetching, storage data
